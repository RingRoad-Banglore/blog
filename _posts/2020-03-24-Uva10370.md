---
title: Uva 10370  Above Average
---


## Input

```
It is said that 90% of frosh expect to be above
average in their class. You are to provide a
reality check.
```

## Output

```
The first line of standard input contains an in-
teger C, the number of test cases. C data
sets follow. Each data set begins with an in-
teger, N , the number of people in the class
(1  N  1000). N integers follow, sepa-
rated by spaces or newlines, each giving the
final grade (an integer between 0 and 100) of
a student in the class.

```

## Sample Input

```
For each case you are to output a line giving the percentage of students whose grade is above average,
rounded to 3 decimal places.

```

## Sample Output

```
5
5 50 50 70 80 100
7 100 95 90 80 70 60 50
3 70 90 80
3 70 90 81
9 100 99 98 97 96 95 94 93 91

```
